Binsta!

## Table of Contents

- [Installation](#installation)


## Requirements

- PHP -V 8.1.0^
- composer
- twig --dev
- RedBeans
- apache (or any other server host)

## Installation

- install composer
- update composer
- setup virtual host to serve the public folder
- create a database
- configure mysql connection in helpers.pnp
- run seeder.php
- go to host
- done
